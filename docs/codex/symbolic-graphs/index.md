# Calcul Symbolique & Graphes

Analyse de graphes et manipulation symbolique avec les bibliothèques Python de référence.

## Pourquoi cette catégorie

SymPy pour le calcul symbolique, NetworkX pour les graphes : deux domaines où le pattern matching de Catnip s'applique
sans friction. Classification de noeuds, parcours de graphes, transformation d'expressions -- tout se ramène à du
pattern matching sur des structures.

## Exemples

| Fichier                                      | Module   | Description                                   |
| -------------------------------------------- | -------- | --------------------------------------------- |
| [`networkx_social.cat`](networkx_social.cat) | networkx | Réseau social, centralité, rôles, communautés |

## Exécution

```bash
catnip symbolic-graphs/networkx_social.cat
```
