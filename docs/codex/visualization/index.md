# Visualisation

Dashboards et graphiques avec les bibliothèques Python de visualisation.

## Pourquoi cette catégorie

hvPlot, Bokeh, holoviews : l'écosystème visualisation Python produit du HTML interactif sans serveur persistant. Catnip
génère les données, construit le layout, et sert le résultat en one-shot.

## Exemples

| Fichier                                        | Modules                   | Description                                                |
| ---------------------------------------------- | ------------------------- | ---------------------------------------------------------- |
| [`hvplot_dashboard.cat`](hvplot_dashboard.cat) | hvplot, holoviews, pandas | Dashboard monitoring capteurs (température, humidité, CO2) |

## Exécution

```bash
catnip visualization/hvplot_dashboard.cat
```
