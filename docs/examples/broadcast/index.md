# Broadcast - cas illustratifs

Documentation courte pour un exemple de broadcast appliqué à des cas concrets. Le fichier source est
[`07_cas_illustratifs.cat`](07_cas_illustratifs.cat). Une forme, plusieurs dimensions.

## Ce que cela montre

- calculs de totaux sur listes imbriquées
- conversions numériques avec `.[op]`
- statistiques simples (moyenne, variance)
- normalisation et remises progressives

## Lancer

```bash
catnip docs/examples/broadcast/07_cas_illustratifs.cat
```
